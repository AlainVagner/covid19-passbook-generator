FAQ
What's that project?
This web application is made by EU citizens in order to let you scan your EU Digital COVID certificate (vaccine, test or healing) and generate a passbook from it. You can then import it into your iPhone's wallet, or if you're using an Android phone, import it into an app that support it.
How does it work?
When you scan the QR-code, we are decrypting it, get the data it containt and create the passbook (the file needed to have a card in your wallet). When it's ready, we send the hash of this passbook to our server for signature. For example: 9673a3bd335365fcf8291117dfad86bce8b642b1. This signature is sent back to your phone which generate the final file and let you save it.
Can I see the code?
Of course. You can read all the source code and dig more into how it works on the Github page of the project. You can even contribute if you want!
